Get the absolute neuron index of a model neuron in its population from its relative index in its population. Example: the simulation consists of two population, two neurons each. p_util.integer_linspace(start, end, step_size, b_include_end=True)[source]¶

Get a list of evenly spaced values between start (included) and end. Values will be integers. b_include_end determines whether end will be included. If so, the final value in the list can be larger than end if end > start or smaller than end if end < start.

Parameters

  • start (int) – First value of the returned sequence.

  • end (int or float) – End of the sequence. If b_round_up == True, the final sequence value will be >= end, otherwise <= end.

  • step_size (int) – Integer value of the difference between two consecutive sequence values.

  • b_include_end (bool) – [default=True] If True, the absolute of the final value of the sequence will be the next larger integer >= abs(end), that is divisible by step_size. If False, it will be the next integer <= abs(end), that is divisible by step_size. If end is an integer that is divisible by step_size, it will be the final value if b_include_end == True.

Returns

sequence

Return type

[int]

p_util.subset_neuron_from_monitors(statemons, spikemons, info, idx_nrn_abs)[source]¶

Get membrane potential, spiketimes etc. of a specific model neuron out of brian2 Monitors and return new monitors containing only this neuron. Monitors will be returned as dictionaries and will probably have to be saved and loaded first using p_io.save_monitors & p_io.load_monitors before they can be used in some analysis/plotting functions.

Parameters

  • statemons (list) – list of brian2 StateMonitor-like(!) SimpleNamespace from .pkl file

  • spikemons (list) – list of brian2 SpikeMonitor-like(!) SimpleNamespace from .pkl file

  • info (list) – list of dicts containing additional information about simulation (as created by run_simulation())

  • idx_nrn_abs (int) – absolute index to the neuron to be extracted

Returns

  • statemons_out: list of dictionaries containining statemonitor data for only the single neuron.

  • spikemons_out: list of dictionaries containining spikemonitor data for only the single neuron.

  • info_out: list of dictionaries containing additional information about the simulation

Return type

  • statemons_out: list

  • spikemons_out: list

  • info_out: list
