Dot plot of one or more groups (x-axis).

Parameters

  • values (list) – list of lists of values to be plotted. one sublist per group. first group is at x=0, second at x=1…

  • group_names (list) – list of group names (strings) that are used as x-tick labels. in order of sublists in values

  • x_label (str) – x-axis label

  • y_label (str) – y-axis label

  • value_colors (list or None) – (list of) list(s) of color values, one color for each value. either one sublist per group, or one list that gets duplicated for each group. in that case it must be at least as long as the largest group

  • marker_size (list or int) – (list of) sizes of dots, one int per value OR one int that will be repeated for all points

  • b_paired (bool) – if True, values in sublists of values are considered paired, i.e. values[0][0] <-> values[1][0] … lines are drawn that connect paried values between groups

  • b_mean (bool) – if True, the group means are plotted as horizontal lines

  • b_jitter (bool) – if True, nearby dots are jittered along the x-axis to avoid overlap

  • jitter_thresh (float) – threshold distance between two consecutive (sorted) y-values, below which dots are to be jittered. If None, the threshold will be automatically calculated depending on the data.

  • jitter_offset (float) – amount by which each dot is offset on the x-axis

  • h_ax (matplotlib.axes._subplots.AxesSubplot) – handle to the axis in which the psths are to be plotted. if None (default) create new figure. if a handle is passed, this function does not have return values.

Returns

  • fig: figure handle

  • ax: axis handles, one sublist per figure

Return type

  • fig: matplotlib.figure.Figure

  • ax: matplotlib.axes._subplots.AxesSubplot
