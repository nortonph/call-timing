p_macro.comborect(stat_x, stat_y, b_skip_fully_represented_params=True, fignum=None)[source]¶

Select rectangular subspace in parex2d plot with two clicks (upper left and lower right corner) and print all parameter values in the combinations of simulations contained in that space. Returns ids of those simulations.

Parameters

  • stat_x (str) – name of key in descriptive_stats.json mapped to the x-axis in the plot

  • stat_y (str) – name of key in descriptive_stats.json mapped to the y-axis in the plot

  • b_skip_fully_represented_params (bool) – if True, don’t print parameters for which all values are represented in rect

  • fignum (int or None) – figure number of the figure containing the parex plot to be targeted. If None, will use plt.gcf()

Returns

id_sims_in_rect: list of ids of the simulations that fall within the rectangle in 2d stats space

Return type

id_sims_in_rect: list

p_macro.parex2d(stat_x, stat_y, colorparam=None, i_colorparam=0, mark=None, identity_line=False, labels=True)[source]¶

Mark all simulations with results in PATH in 2d space of the destrciptive stats (.json) given by stat_x & stat_y.

Parameters

  • stat_x (str) – name of key in descriptive_stats.json to be mapped to the x-axis

  • stat_y (str) – name of key in descriptive_stats.json to be mapped to the y-axis

  • colorparam (str) – name of explored parameter, whose values should be used as a color code for dots/text

  • i_colorparam (int) – index to parameter given by colorparam (i.e. population/synapse index)

  • mark (int or list or None) – (list of) sim_id(s) to highlight on plot

  • identity_line (bool) – if True, add an identity line (line through x==y)

  • labels (bool) – if True, add text labels of simulation ids next to points

Returns

id_sims_all: list of ids of all simulations

Return type

id_sims_all: list

p_macro.trace(id_sims, sortby=None, hline=None, xlims=None, figsize=(16, 9))[source]¶

plots trace of neuron of interest (as defined by config[‘plot’]) of the parex simulation file in PATH, identified by id_sims (e.g. id_sims==23 => opens PATH/parex_000023.pkl)

Parameters

  • id_sims (int or list) – (list of) id(s) of simulation(s) in parameter exploration, contained in .pkl file name

  • sortby (str or None) – name of descriptive stats parameter by which to sort the subplots, e.g. ‘t_spike1’

  • hline (int or float) – y-value of gray horizontal line to add to each subplot

  • xlims (tuple or list) – x-axis limits (lower, upper)

  • figsize (tuple or list) – size of figure in inches (width, height)

Returns
